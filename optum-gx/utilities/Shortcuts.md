# Shortcuts

A reference of keyboard and mouse shortcuts available in OPTUM GX.

## Analysis

| Shortcut | Action |
| -------- | ------ |
| **F5**   | Run analysis |

## Editing

| Shortcut | Action |
| -------- | ------ |
| **Ctrl + C** | Copy |
| **Ctrl + V** | Paste |
| **Ctrl + X** | Cut |
| **Ctrl + Z** | Undo |
| **Ctrl + Y** | Redo |
| **Ctrl + S** | Save project |

## Mouse

| Action | Input |
| ------ | ----- |
| **Select** | Left mouse button |
| **Multi-select** | Shift + left mouse button |
| **Pan** | Middle mouse button |
| **Zoom** | Mouse wheel |
