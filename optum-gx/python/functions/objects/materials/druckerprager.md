# DruckerPrager

Drucker-Prager elastoplastic material -- smoothed approximation of the
Mohr-Coulomb failure surface, well-suited to 3D analyses.

## Examples

```python
material = project.DruckerPrager(name='DP Sand', M=1.0)
material.M
```
