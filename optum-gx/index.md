# OPTUM GX Documentation

Welcome to the **OPTUM GX** knowledge base.

This documentation covers both the theoretical background and practical use cases of OPTUM GX, including:

- Material models and constitutive relations
- Numerical analysis workflows and modeling concepts
- Python scripting and automation
- Worked examples and tutorials

Whether you are building models through the graphical interface or automating workflows with Python, this knowledge base is designed to help you get started and as reference work to help bridge knowledge gaps when working with the program.

---

## Documentation Areas

### Materials
Learn how material models in OPTUM GX are formulated and configured, including constitutive behavior, parameters, and analysis considerations. [Materials](./materials/1-notation/)

### Python
Automate and extend OPTUM GX using Python. The Python interface provides direct access to projects, models, analyses, and results for custom workflows, parametric studies, and even customizable plugins. [Python](./python/)
