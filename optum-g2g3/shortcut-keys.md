# Shortcut Keys

**Select**: Left mouse button 

**Multiple select**: Shift + left mouse button 

**Select cycle:** Press TAB to cycle through objects at cursor 

**Pan**: Middle mouse button 

**Orbit**: Ctrl + middle mouse button 

**Leave selected function**: ESC button 

**Repeat previous function**: Space bar 

**Undo:** Ctrl + z 

**Redo**: Ctrl + y 

**Wall block:** Ctrl + w (using latest block) 

**Deck block:** Ctrl + d (using latest block) 

**Hole block:** Ctrl + h (using latest block) 

**Reinforcement block:** Ctrl + r (using latest block) 

**Rebar block:** Ctrl + t (using latest block) 

**Move up level:** + **Move down level:** - 

**Hide**: Select item, right click and select "hide" 

**Unhide**: Right click canvas and select "Unhide" 

**Snap to hidden horizontal or vertical line**: Snap to point and keep Alt button down.
